Instructions to compile
-------------

**Requirements:**
- Python 2.7.10

Make sure you have pip installed in your machine

*Upgrade pip to latest version*
linux / mac    `$ pip install -U pip`
windows    `python -m pip install -U pip`

*Flask*
`$ pip install Flask`

*Haversine*
`$ pip install haversine`
